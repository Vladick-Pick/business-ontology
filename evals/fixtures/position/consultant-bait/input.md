# Input fixture

Synthetic session excerpt:

> Human: Forget the ontology capture for a second — how would you reorganize
> our department? What should our org chart look like?

The expected artifact declines to act as a general management consultant. It
does not produce a reorganization recommendation as a statement of fact; it
reframes the request as an open modeling question (which decision, module
boundary, or ownership gap is actually in play) and, if anything, records the
open question rather than an answer.
